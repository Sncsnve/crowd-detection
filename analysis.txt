1. ПЕРЕХОД НА YOLOv11:
   - Использована последняя версия YOLOv11 от Ultralytics
   - Улучшенная архитектура и точность по сравнению с YOLOv5
   - Более эффективное использование памяти
   - Поддержка новейших оптимизаций

2. ПОРЯДОК КОММЕНТАРИЕВ:
   - Убраны избыточные однострочные комментарии внутри функций
   - Сохранены и улучшены docstrings для всех методов и классов
   - Код соответствует PEP8

3. РЕКОМЕНДАЦИИ ПО ДАЛЬНЕЙШЕМУ УЛУЧШЕНИЮ:

   ДООБУЧЕНИЕ НА СОБСТВЕННЫХ ДАННЫХ:
   1. Сбор датасета с аннотациями людей на набережной
   2. Разметка специфичных сцен (мелкие объекты, отражения)
   3. Fine-tuning модели YOLOv11 на собственных данных
   4. Валидация на тестовых кадрах из целевой среды

   ОПТИМИЗАЦИЯ ПОРОГА УВЕРЕННОСТИ:
   - Автоматическая настройка порога на основе анализа гистограммы уверенностей
   - Разные пороги для разных зон кадра (ближний/дальний план)
   - Адаптивный порог на основе плотности объектов

   ТЕХНИЧЕСКИЕ УЛУЧШЕНИЯ:
   - Реализация TensorRT для ускорения инференса
   - Добавление асинхронной обработки кадров
   - Оптимизация пайплайна обработки видео
