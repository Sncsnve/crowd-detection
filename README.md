Проект для тестового задания на позицию Junior Data Scientist. Программа выполняет детекцию людей на видео с использованием модели YOLOv11, отрисовывает bounding boxes и сохраняет результат.

Результаты работы: Выходное видео с детекцией [crowd_output.mp4](https://drive.google.com/drive/folders/113iUzMGb2zHBkPc05e6u1Xn15bR7Iea3?usp=sharing)

Статистика обработки:
Обработано кадров: 705
Всего обнаружено людей: 5294
Среднее людей на кадр: 7.51
Максимум людей в кадре: 11
FPS обработки: 4.28
Время обработки: 164.89 сек

Инструкцией по воспроизведению результата:
1. Скачать проект с GitHub
2. Открыть командной строкой и выполнить команду:
git clone https://github.com/Sncsnve/crowd-detection.git
После этого перейти в созданную папку командой:
cd crowd-detection
3. Установить все необходимые библиотеки. Выполнить команду:
pip install -r requirements.txt
4. Перед запуском перенести исходный видеофайл crowd.mp4 в папку с проектом
5. Запустить программу. Выполнить команду: python main.py
Программа найдёт видео crowd.mp4, обработает его и сохранит результат в папку output
